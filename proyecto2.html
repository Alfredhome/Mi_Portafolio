Analizador Léxico

Este proyecto es una aplicación de escritorio diseñada para analizar léxicamente fragmentos de código fuente escritos en un lenguaje de programación específico. Esta herramienta es capaz de escanear y analizar sintácticamente el código, identificar tokens y generar un archivo Excel que presenta una representación estructurada de los tokens y su información asociada.

Caracteristicas :

  1. Escaneo de Código: Los usuarios pueden escribir o cargar código fuente desde un archivo externo. El analizador léxico puede procesar múltiples líneas de código, separando el texto en tokens individuales y determinando su categoría (identificadores, palabras clave, símbolos, literales, etc.).
  2. Análisis Léxico: El núcleo del proyecto es el analizador léxico, que reconoce y clasifica cada token del código. Puede identificar palabras clave del lenguaje, operadores, símbolos y otros elementos léxicos relevantes.
  3. Exportación a Excel: Los informes generados se pueden exportar a archivos Excel, lo que facilita su distribución y análisis posterior.
  4. Gestión de Archivos: La aplicación permite guardar y cargar configuraciones de análisis y archivos de código para un uso posterior.

Tecnologias Usadas

  • C#: Lenguaje de programación principal para el desarrollo de esta aplicación

  • Interoperabilidad de Excel: Utilización de bibliotecas o enfoques para la generación y manipulación de archivos Excel.

proporciona una herramienta útil para los desarrolladores y estudiantes que desean comprender y analizar la estructura léxica de los fragmentos de código fuente en diversos lenguajes de programación.
